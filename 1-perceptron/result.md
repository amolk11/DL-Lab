# Perceptron Experiment Results

## Dataset
Synthetic linearly separable dataset generated using NumPy.

## Observations

1. Perceptron converges quickly for linearly separable data.
2. Decision boundary becomes stable after a few epochs.
3. Learning rate influences convergence speed.

## Limitations

Perceptron fails when data is not linearly separable.

## Future Experiments

- Test perceptron on non-linearly separable data
- Compare with logistic regression
